### 4.1 Financial Health Scorecard (Graham-Inspired)

**Category 1: Earnings Quality (25 points)**
- 10-year EPS CAGR > 10% → 10 pts
- EPS Consistency (no loss year in 10 yrs) → 8 pts
- Quality of Earnings (CFO/Net Income > 0.9) → 7 pts

**Category 2: Balance Sheet Strength (25 points)**
- Current Ratio > 2 → 8 pts
- Debt/Equity < 0.5 → 10 pts
- Interest Coverage > 5x → 7 pts

**Category 3: Profitability & Returns (25 points)**
- ROE > 15% (5-year avg) → 10 pts
- ROCE > 15% (5-year avg) → 8 pts
- Net Profit Margin trend (improving) → 7 pts

**Category 4: Dividend & Capital Allocation (25 points)**
- Dividend paid for 5+ years → 8 pts
- Payout Ratio (20–50%) → 7 pts
- Buybacks or value-accretive acquisitions → 10 pts

**Total: 100 Points → Grade A (80–100), B (60–79), C (40–59), D (<40)**

## PART 5: VALUATION ENGINE

### 5.1 Multi-Model Valuation (Buffett Approach)

**Model 1: Discounted Cash Flow (DCF)**
```
Intrinsic Value = Σ [FCFt / (1+WACC)^t] + Terminal Value
- FCF Growth (Phase 1: 3 yrs): Historical FCF CAGR
- FCF Growth (Phase 2: 4-7 yrs): Conservative normalized
- Terminal Growth Rate: India long-run GDP (~6.5%)
- WACC: Risk-free rate (10yr G-Sec) + Beta × Equity Risk Premium
```

**Model 2: Graham Number**
```
Graham Number = √(22.5 × EPS × Book Value Per Share)
Buy Zone: Market Price < Graham Number × 0.7 (30% MoS)
```

**Model 3: Earnings Power Value (EPV)**
```
EPV = Normalized EBIT × (1 - Tax Rate) / WACC
Compares EPV with Asset Value to identify moat
```

**Model 4: Dividend Discount Model (DDM)**
```
For dividend-paying blue chips:
P = D1 / (Ke - g)
Where g = Sustainable Growth Rate = ROE × Retention Ratio
```

**Model 5: Relative Valuation**
```
- Sector P/E comparison (vs NSE sector index)
- EV/EBITDA vs 5-year median
- P/B vs sector peers
```

**Final Intrinsic Value = Weighted Average of All Models**
```
Final IV = 0.40×DCF + 0.25×EPV + 0.20×Relative + 0.10×Graham + 0.05×DDM
```

## PART 6: MOAT ANALYSIS ENGINE (Buffett Framework)

### 6.1 Competitive Moat Scoring (100 points)

**1. Cost Advantage (20 pts)**
- Lowest cost producer in sector
- Economies of scale evidence
- Operational efficiency vs peers

**2. Network Effect (20 pts)**
- Platform-based business (exchanges, marketplaces)
- User growth → value growth loop

**3. Switching Costs (20 pts)**
- Enterprise software / B2B contracts
- Long-term regulatory licensing
- High customer retention rate (>90%)

**4. Intangible Assets (20 pts)**
- Brand strength (survey-based or market share)
- Patents, proprietary technology
- Regulatory licenses (BFSI, pharma, telecom)

**5. Efficient Scale (20 pts)**
- Duopoly or oligopoly market position
- High capex barrier for new entrants
- Market share stability over 5 years

**Moat Rating: Wide (80-100) | Narrow (50-79) | None (<50)**

## PART 16: SAMPLE PYTHON CODE STRUCTURE

```python
# Master Agent Orchestrator
class NSEForecastAgent:
    def __init__(self, ticker: str):
        self.ticker = ticker
        self.fundamental_agent = FundamentalAnalysisAgent(ticker)
        self.valuation_agent = ValuationAgent(ticker)
        self.moat_agent = MoatAnalysisAgent(ticker)
        self.management_agent = ManagementQualityAgent(ticker)
        self.sentiment_agent = MarketSentimentAgent(ticker)
        self.geo_agent = GeopoliticalRiskAgent(ticker)
        self.industry_agent = IndustryTrendAgent(ticker)
        self.technical_agent = TechnicalAnalysisAgent(ticker)

    def run_analysis(self) -> StockReport:
        # Run all sub-agents in parallel
        results = asyncio.gather(
            self.fundamental_agent.analyze(),
            self.valuation_agent.analyze(),
            self.moat_agent.analyze(),
            self.management_agent.analyze(),
            self.sentiment_agent.analyze(),
            self.geo_agent.analyze(),
            self.industry_agent.analyze(),
            self.technical_agent.analyze()
        )
        return self.synthesize_report(results)

    def synthesize_report(self, results) -> StockReport:
        composite_score = (
            results.fundamental.score * 0.25 +
            results.valuation.score   * 0.20 +
            results.moat.score        * 0.20 +
            results.management.score  * 0.15 +
            results.sentiment.score   * 0.08 +
            results.geo_risk.score    * 0.07 +
            results.industry.score    * 0.05
            # Technical used only for entry timing
        )
        recommendation = self.get_recommendation(composite_score, results.valuation)
        return StockReport(ticker=self.ticker, score=composite_score, **recommendation)
```

## APPENDIX A: KEY FORMULAS REFERENCE

```
Graham Number          = √(22.5 × EPS × BVPS)
Net Current Asset Val  = Current Assets − Total Liabilities
Owner Earnings         = Net Income + D&A − Maintenance Capex
WACC                   = Ke×(E/V) + Kd×(1-T)×(D/V)
Sustainable Growth     = ROE × (1 − Dividend Payout Ratio)
Earnings Yield         = EPS / Market Price (compare vs G-Sec yield)
PEG Ratio             = P/E ÷ EPS Growth Rate (Buffett: PEG < 1 ideal)
Economic Profit       = NOPAT − (Invested Capital × WACC)
```
